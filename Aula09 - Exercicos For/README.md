# REalizando exercicio em sala de aula

## Exercicios de For.

1. Solicite ao usuário uma quantidade limite e exiba em tela a repetição de 1 até o limite.
1. Solicite ao usuário uma quantidade inicial e final, para exibir em tela a repetição do inicio até o final
1. Solicite ao usuário um valor numero para exibir a contagem regressiva até o zero
1. Retorne a tabuada do numero escolhido
1. Retornar todas as tabuadas de 1 até a tabuada do numero inf
